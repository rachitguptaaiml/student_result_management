Student Result Management System
==============================

A simple SQL-based system to manage and analyze student results.

### Features:

*   Manages exam scores, subjects, and students.
*   Calculates average scores for each subject per student.
*   Displays the results in a user-friendly format.

### Setup/Run Instructions:

1.  Install the required dependencies by running `pip install -r requirements.txt`.
2.  Create a new SQLite database named `student_results.db` if it doesn't exist already.
3.  Run the script using Python: `python main.py`

### Sample Output:
| Name | Subject      | Average Score |
| --- | ------------ | -------------- |
| Student 1 | Math         | 18.333333      |
| Student 2 | Science      | 19.500000      |
| ...     | ...          | ...            |
